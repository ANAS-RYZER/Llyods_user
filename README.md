# Ryzer Web application

A modern web application built with **Next.js**, **TailwindCSS**, **ShadCN UI components**, and **TypeScript**. This project provides a solid foundation for developing scalable, fast, and accessible web applications.

## üöÄ Features

- **Next.js**: A React framework with server-side rendering (SSR) and static site generation (SSG) capabilities.
- **TailwindCSS**: A utility-first CSS framework for fast UI development.
- **ShadCN UI**: Beautiful and accessible UI components for seamless design integration.
- **TypeScript**: Strongly typed JavaScript for fewer bugs and better developer experience.
- **ESLint & Prettier**: Code linting and formatting tools for clean, maintainable code.
- **Performance Optimization**: Leveraging Next.js features like image optimization and dynamic imports.

## üì¶ Getting Started

### Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (LTS version recommended)
- [npm](https://npm.io/) (or your preferred package manager)

### Installation

1. Clone the repository:

   git clone https://github.com/durga-altrova-xyz/fandora-web.git
   cd ryzer-user-v2

2. install packages 
  : npm install --legacy-peer-deps
3. dev command
  : npm run dev
   
5. build command
  : npm run build
